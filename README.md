**AIR AI** is a sophisticated mobile chatbot application built on top of the **MIT App Inventor** framework. It leverages the power of Google's **Gemini 1.5 Flash API** to provide real-time, intelligent, and context-aware responses within a high-performance, modern user interface.

---

## 🎨 Design Philosophy: "Airy & Modern"
Inspired by the lightness of air, the UI features a **Glassmorphism** aesthetic. 
- **Sleek Interface:** Neon-cyan accents on a deep-charcoal background.
- **Responsive Layout:** Optimized for various Android screen sizes.
- **Futuristic UX:** Includes typing animations and dynamic status monitoring.

## 🛠️ Technical Specifications
* **Core Model:** Google Gemini 1.5 Flash (via REST API).
* **Architecture:** Hybrid Web-App Integration (WebViewer + JavaScript Bridge).
* **Security:** Port 443 HTTPS encrypted communication.
* **Asset Pipeline:** Custom-designed vector graphics & splash screens.

## 🚀 How to Setup (.aia instructions)

To get this project running on your local machine:

1.  **Download** the `AIR_AI.aia` file from this repository.
2.  **Import** the file into [MIT App Inventor](http://ai2.appinventor.mit.edu/).
3.  **API Configuration:**
    * Open the `Blocks` editor.
    * Find the `API_KEY` variable.
    * Replace it with your own key from [Google AI Studio](https://aistudio.google.com/).
4.  **Build:** Generate the `.apk` and enjoy your personal AI assistant!
